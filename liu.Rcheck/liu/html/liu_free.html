liu_free {liu}R Documentation

Free LIU Index Memory

Description

Explicitly releases the memory allocated for the LIU index in C. Use this when index is no longer needed to prevent memory leaks.

Usage

liu_free(index)

Arguments

index

LIU index object (external pointer).

Examples

## Not run: 
idx <- liu_build(mtcars, "hp")
# ...
liu_free(idx)

## End(Not run)

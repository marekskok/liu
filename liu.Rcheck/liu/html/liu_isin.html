liu_isin {liu}R Documentation

Is In LIU Index

Description

Checks if given keys are present in LIU index. For NA it returns FALSE.

Usage

liu_isin(index, keys)

Arguments

index

A LIU index object (external pointer).

keys

Vector of keys (int or double must match LIU index type) to check.

Value

Vector of logical values.

Examples

## Not run: 
Checks in int index if 2, NA, 5 are present
logical <- liu_isin(idx, as.integer(c(2,NA,5)))

Checks in double index if 6.7 is present
logical <- liu_isin(idx, 6.7)

## End(Not run)

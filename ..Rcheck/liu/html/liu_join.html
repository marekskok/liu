liu_join {liu}R Documentation

Fast Inner Join

Description

Performs a high-performance Join between two data frames using a LIU index. For now only inner (default) and left join are available. It takes 4 arguments: left data table, name of one of it's columns, right data table, index built on it. Type of chosen column must match index type. Indexes doesn't take NA, so NA in given column are ignored. It returns the same data frame as merge(df_left, df_right, "id", incomparables = NA)

Usage

liu_join(df_left, column_name, df_right, index, how = "inner")

Arguments

df_left

Data frame (left side of the join).

column_name

Name of the join numeric column (must exist in left data frames).

df_right

The "indexed" data frame (right side of the join).

index

A LIU index object built on the join column of 'df_right'.

how

A character string "inner" or "left".

Value

Returns merged data frame.

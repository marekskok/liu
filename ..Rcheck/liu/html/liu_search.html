liu_search {liu}R Documentation

Search for Keys in LIU Index

Description

Performs a fast lookup in the LIU index to find all row indices associated with given vector of keys. It ignores NA in keys vector.

Usage

liu_search(index, key)

Arguments

index

LIU index object (external pointer).

key

Vector of keys (int or double must match LIU index type) to search for.

Value

Integer vector of row indices where the keys were found. Returns an empty vector if none of the keys exist.

Examples

## Not run: 
row_idx <- liu_search(idx, 110)
df[row_ids, ]

row_ids <- liu_search(idx, c(6.7,21.15))
df[row_idx, ]

## End(Not run)
